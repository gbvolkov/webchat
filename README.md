# GWP Chat

Переиспользуемый чат компонент для Vue 3 проектов с поддержкой TypeScript и современными инструментами разработки.

## Архитектура

Проект следует принципам Domain-Driven Design (DDD) для разделения UI и бизнес-логики:

```
src/
├── domain/           # Бизнес-логика
│   ├── chat/         # Чат доменная модель
│   │   ├── api/      # API методы
│   │   ├── composables/ # Vue композиции
│   │   └── models/   # TypeScript типы
│   └── config/       # Конфигурация
└── ui/               # UI компоненты
    ├── common/       # Общие компоненты
    ├── page/         # Компоненты для отдельной страницы
    └── widget/       # Виджет компоненты
```

## Требования системы

⚠️ **ВАЖНО**: Для работы проекта требуется Node.js **20.19.0** или выше!

- **Node.js**: `20.19.0+` (обязательно!)
- **Vue**: `3.5+`
- **TypeScript**: `5.9+`

## Быстрая установка

### Для разработчиков команды
1. **Проверьте версию Node.js**: `node --version` (требуется 20.19.0+)
2. **Если версия < 20.19.0** - обновите Node.js
3. **Установите зависимости**: `npm install`
4. **Запустите проект**: `npm run dev`

## Развертывание

### Скрипты

```bash
# Разработка
npm run dev

# Сборка
npm run build

# Проверка типов
npm run type-check

# Линтинг
npm run lint         # с автоисправлением
npm run lint:check   # только проверка

# Форматирование
npm run format       # применить форматирование
npm run format:check # проверить форматирование
```
